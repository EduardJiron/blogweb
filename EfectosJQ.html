Selectores jQuery

Los selectores jQuery le permiten seleccionar y manipular elementos HTML.
Los selectores jQuery se utilizan para "encontrar" (o seleccionar) elementos
HTML en función de su nombre, id, clases, tipos, atributos, valores de atributos
y mucho más. Se basa en los selectores CSS existentes y, además, tiene algunos
selectores personalizados propios.
Todos los selectores en jQuery comienzan con el signo de dólar y paréntesis: $().

El selector de elementos jQuery selecciona elementos en función del nombre del elemento.
