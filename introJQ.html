¿Qué es jQuery?

JQuery es una biblioteca de JavaScript ligera, "escribe menos, hace más".
El propósito de jQuery es hacer que sea mucho más fácil usar JavaScript en su sitio web.
jQuery toma muchas tareas comunes que requieren muchas líneas de código ne
JavaScript para realizarlas y las envuelve en métodos a los que puede
llamar con una sola línea de código.

La biblioteca jQuery contiene las siguientes características:

-Manipulación HTML/DOM.
-Manipulación de CSS.
-Métodos de eventos HTML.
-Efectos y animaciones.
-AJAX.
-Utilidades
